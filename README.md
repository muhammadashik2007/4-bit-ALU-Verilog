# 4-bit-ALU-Verilog
# 4-bit ALU using Verilog

## Description
This project implements a 4-bit Arithmetic Logic Unit (ALU) in Verilog HDL.

## Features
- 4-bit Addition
- 4-bit Subtraction (using Two's Complement)
- AND
- OR
- XOR
- NOT

## Inputs
- A[3:0]
- B[3:0]
- OP[2:0]

## Outputs
- Result[3:0]
- Carry
- Zero

## Tools Used
- Verilog HDL
- Icarus Verilog
- EDA Playground

## Operations

| OP | Operation |
|----|-----------|
|000|Addition|
|001|Subtraction|
|010|AND|
|011|OR|
|100|XOR|
|101|NOT|
